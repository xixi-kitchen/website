# HUGH·Aix 个人网站

一个现代化的个人作品集网站，展示设计、开发与创新。

## 技术栈

- **前端框架**: Next.js
- **样式方案**: Tailwind CSS
- **动画效果**: Framer Motion, GSAP
- **字体**: MiSans VF, PingFang SC, Nabla
- **部署**: Vercel

## 主要功能

- 🎨 响应式设计，完美适配移动端和桌面端
- 🌓 支持亮色/暗色模式切换
- ✨ 流畅的页面过渡动画
- 🎯 滚动触发的渐入动画
- 🎭 优雅的 UI 设计
- 🌐 多语言支持（中/英）

## 页面结构

- **首页**: 个人简介和主要作品展示
- **AI**: AI 技术研究与应用
- **经历**: 个人成长历程
- **项目**: 项目作品集
- **创意**: 创意实验空间
- **联系**: 联系方式
- **关于**: 关于我

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 启动生产服务器
npm start
```
